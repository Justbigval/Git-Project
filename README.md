# React Native App - Solomon Clifford Seyi

Welcome to the Expo Calculator application repository for the Git & Mobile App assignment!

## Implementation Details

- **Responsive UI:** Adapts cleanly to both iOS and Android screens.
- **Styling:** Custom theme styling overrides applied via a centralized theme file.
- **Expo Architecture:** Bootstrapped via `create-expo-app` with routing features.

## Testing locally

1. Run `npm install` to download all React Native packages.
2. Run `npm start` to spin up the Expo server.
3. Open the app via Expo Go.
